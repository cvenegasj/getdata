# Description of the script functioning
It is necessary to import reshape2 and dplyr libraries for the script to work correctly.
Here are the steps executed by the script

1. Reading data sets.
2. Combining data sets.
3. Setting names for columns.
4. Reading descriptive activities data.
5. Reading subjects data.
6. Deleting columns which give us problems for selecting only the 'mean' and 'std' columns.
7. Selecting only the 'mean' and 'std' columns for our purpose.
8. Adding an 'activity' and 'subject' column from the step 4 and 5.
9. Giving a descriptive name for activities.
